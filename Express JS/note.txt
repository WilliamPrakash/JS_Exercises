This folder's purpose is to highlight the main features/building blocks of
Express JS

Express JS is a popular framework for Node.js. It's main uses include 
building web apps and APIs.

Express helps simplify routes, setting up ports, and HTTP request handling

Express uses middleware: stuff that runs between the request being sent to
your server, and the actual response being returned to the user
